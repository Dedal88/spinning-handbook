# Справочник-спиннингиста

Данный ресурс предназначен для любителей спиннинговой ловли, а также тем, кто только начинает осваивать эту деятельность.

Прогнозирование клёва рыбы основывается на:

- Народных календарях клёва
- Информации о поведении рыбы из литературы, посвящённой рыбной ловле
- Личных наблюдениях автора

**Важно:** Этот прогноз является субъективным и не претендует на стопроцентную точность.

### Заполнение полей

- **"Кого ловить будем ?!"** — обязательное поле
- **"Когда ловить будем ?!"** — необязательное поле (если не заполнить, информация о клёве не будет выведена)
- **"Где ловить будем ?!"** — необязательное поле (если не заполнить, информация о водоёмах не будет выведена)

После заполнения необходимых полей нажмите кнопку **"Получить информацию"**.

## О проекте

**Справочник-спиннингиста** — это веб-приложение для прогнозирования клёва рыбы и предоставления информации о водоёмах. Приложение основано на народных календарях клёва, литературе по рыбной ловле и личных наблюдениях автора.

### Функционал

- Выбор вида рыбы для ловли
- Прогноз клёва по месяцам
- Информация о поведении рыбы и местах обитания
- Список водоёмов в Ивановской и Ярославской областях
- Адаптивный дизайн для мобильных устройств

## Технологический стек

- **HTML5** — семантическая разметка
- **CSS3** — стилизация с использованием Flexbox 
- **JavaScript (ES6+)** — логика приложения
- **Webpack** — сборка проекта

## Установка и запуск

### Предварительные требования

- Node.js (версия 14 или выше)
- npm (версия 6 или выше)

### Клонирование репозитория

```
git clone https://github.com/Dedal88/spinning-handbook.git
cd spinning-handbook
```

### Установка зависимостей

```
npm install
```

### Запуск в режиме разработки

```
npm run dev
```

Приложение будет доступно по адресу `http://localhost:8080`

### Сборка для продакшена

```
npm run build
```

Собранные файлы будут находиться в папке `dist`

### Развёртывание на GitHub Pages

```
npm run deploy
```

## Важное замечание

Прогноз клёва является **субъективным** и не претендует на стопроцентную точность. Рекомендуется использовать информацию как справочную и учитывать реальные погодные условия и особенности конкретного водоёма.
